# Typography Guidelines

## Type Scales
Select a set of sizes that work well rather than using mathematical ratios that produce fractional pixels.
Font sizing should use `px` or `rem` units to avoid the unpredictable compounding effects of `em` units in nested elements.

Recommended scale: 12, 14, 16, 18, 20, 24, 30, 36px

## Font Selection
Prioritize neutral sans-serif typefaces and choose fonts offering multiple weights.
Avoid condensed fonts with limited x-heights for body text.
Leverage popular options from sources like Google Fonts.

## Line Height
Line height varies inversely with font size:
- Body text: ~1.5–1.7
- Headings: ~1.0–1.2
Optimal line length: 45–75 characters per line.
Wider columns benefit from increased line height.

## Alignment
- Left alignment: standard for readability.
- Justified text: avoid on web without hyphenation.
- Center alignment: only for headlines or minimal text.
- Tabular data: right-align numbers.
- Mixed font sizes on one line: align at baseline, not vertical center.

## Letter Spacing
- Display text: slightly tightened spacing.
- All-caps text: increased tracking for readability.
